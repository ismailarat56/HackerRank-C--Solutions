# HackerRank-C--Solutions

Bu depo, HackerRank platformunda çözdüğüm algoritma ve veri yapıları problemlerinin **C++** kodlarını içermektedir. 

## Çözülen Sorular
1. Solve Me First
2. Simple Array Sum
3. A Very Big Sum
4. Diagonal Difference


Not: Bu depodaki dosyalarda yer alan main fonksiyonları ve standart veri okuma/yazdırma işlemleri HackerRank tarafından şablon olarak verilmiştir. Problemleri çözen ana algoritma fonksiyonları tarafımca yazılmıştır.
